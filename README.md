# NASA-API
The NASA Astronomy Picture of the Day fetched through the NASA API and displayed on this Web Application. 

# Technologies Used:
- API Fetch
- HTTP
- JSON
- DOM Manipulation (to display the JSON data!)
- JavaScript ES6+
- Async Await
- UI/UX
- SCSS (SASS)
- CSS Flexbox
- HTML
