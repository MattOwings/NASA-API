# NASA-API

This Web Application accesses the NASA API, Fetches JSON data, and displays the data.

# Technologies Used:
- API Fetch / Response
- JSON
- DOM Manipulation (to display the JSON data!)
- JavaScript ES6+
- Async Await
- UI/UX
- SCSS (SASS)
- CSS Flexbox
- HTML
