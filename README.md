# NASA-API

This Web Application uses the NASA API, Fetches JSON data through an HTTP Request, and displays the data on the front-end. 

# Technologies Used:
- API Fetch
- HTTP
- JSON
- DOM Manipulation (to display the JSON data!)
- JavaScript ES6+
- Async Await
- UI/UX
- SCSS (SASS)
- CSS Flexbox
- HTML
